# Glossary

## A

**APR (Annual Percentage Rate)**
The yearly interest rate without compounding.

**APY (Annual Percentage Yield)**
The yearly interest rate with compounding included.

## B

**BPS (Basis Points)**
One hundredth of a percent. 100 BPS = 1%.

**BSC (BNB Smart Chain)**
The blockchain LEVERAGED is built on. Formerly Binance Smart Chain.

## C

**Collateral**
Assets deposited as security for a leveraged position.

**Chainlink**
Decentralized oracle network providing price feeds.

## D

**DeFi (Decentralized Finance)**
Financial services built on blockchain without intermediaries.

## E

**Entry Fee**
The 0.1% fee charged when opening a position.

## H

**Health Factor**
A measure of position safety. Position Value / Debt. Below 1.1 = liquidatable.

## K

**Keeper**
A bot or user authorized to execute liquidations.

## L

**Leverage**
Using borrowed funds to increase position size. 5x leverage means $1 controls $5 of exposure.

**Liquidation**
Forced closure of a position when health factor drops below threshold.

**Liquidity**
Available funds in the lending pool for borrowing or withdrawal.

**LP (Liquidity Provider)**
Someone who deposits funds into a liquidity pool.

**LVG**
The native token of LEVERAGED protocol.

## O

**Oracle**
A service providing external data (like prices) to smart contracts.

## P

**Position**
An open leveraged trade tracking an asset's price movement.

**PnL (Profit and Loss)**
The gain or loss on a position.

## S

**Slippage**
The difference between expected and executed price.

**Smart Contract**
Self-executing code on the blockchain.

**Staking**
Locking tokens to earn rewards or benefits.

## T

**TVL (Total Value Locked)**
Total assets deposited in the protocol.

## U

**USDT**
Tether USD stablecoin, used as collateral in LEVERAGED.

**Utilization Rate**
Percentage of deposited funds currently borrowed.

## V

**Value Increase Fee**
The 25% fee on profits when closing a position.

**Vault**
Smart contract holding user funds and managing positions.

## W

**Wallet**
Software or hardware for storing cryptocurrency private keys.

**Wei**
Smallest unit of ETH/BNB. 1 token = 10^18 wei.
