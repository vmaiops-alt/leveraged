## Planned Rewards

| Severity | Reward | Examples |
|----------|--------|----------|
| Critical | Up to $50,000 | Fund theft, permanent DoS |
| High | Up to $20,000 | Temporary fund lock, griefing |
| Medium | Up to $5,000 | Economic attacks, fee bypass |
| Low | Up to $1,000 | Minor issues, gas inefficiency |

## Scope

### In Scope
- All deployed smart contracts
- Frontend security issues
- API vulnerabilities

### Out of Scope
- Already known issues
- Theoretical attacks without PoC
- Social engineering
- Third-party services
- Testing on mainnet without permission

## Rules

1. **No public disclosure** until fixed
2. **Provide clear PoC** demonstrating the issue
3. **One submission per issue**
4. **First reporter gets the bounty**
5. **No testing on mainnet** without coordination

## How to Report

1. Email: security@leveraged.finance (planned)
2. Include:
   - Detailed description
   - Steps to reproduce
   - Proof of concept
   - Suggested fix (optional)

## Response Time

| Stage | Timeline |
|-------|----------|
| Acknowledgment | 24 hours |
| Initial assessment | 3 days |
| Fix development | 7-14 days |
| Bounty payment | 30 days |

## Safe Harbor

We will not pursue legal action against researchers who:
- Follow responsible disclosure
- Don't exploit beyond PoC
- Don't harm users or protocol
- Give us reasonable time to fix
