## Colors

### Primary Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Indigo | `#6366F1` | Primary actions, links |
| Green | `#22C55E` | Success, profits |
| Red | `#EF4444` | Danger, losses |
| Yellow | `#EAB308` | Warning |

### Dark Theme

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#11111B` | Main background |
| Surface | `#181825` | Cards, panels |
| Border | `#1E1E2E` | Dividers |

## Typography

**Primary Font:** Inter

- Headings: Inter Bold
- Body: Inter Regular
- Code: JetBrains Mono

## Usage Guidelines

### Do ✅
- Use official logos
- Maintain aspect ratio
- Use on appropriate backgrounds
- Link to official channels

### Don't ❌
- Modify colors or proportions
- Use low-resolution versions
- Imply partnership without permission
- Use in misleading context
